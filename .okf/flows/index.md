# Flows

* [Core pipeline](core-pipeline.md) - the whole capture → distill → inject → gate picture in one file.
* [Capture](capture.md) - session Stop event → raw dump.
* [Distill](distill.md) - raw dumps → knowledge notes + proposals.
* [Inject](inject.md) - knowledge notes → live session context.
* [Proposal approval](proposal-approval.md) - the human gate: proposal → scaffolded skill.
* [Backfill](backfill.md) - recovers pre-plugin sessions into the vault.
* [Dashboard](dashboard.md) - renders the vault as a self-contained HTML report.
* [Reset](reset.md) - deletes or re-queues vault data on request.
